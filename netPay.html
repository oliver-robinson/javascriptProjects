Net Pay Program

Net pay calculator that takes your GROSS pay and then deducts Federal, State, and FICA. It has option to deduct additional amount from your check if you choose to.

  1. Start
  2. Display, "How much did you earn a year?"
  3. Input grossPay
  4. Display, "Give me a second while I calculate your check!"
  5. Set FED = .15
  6. Set STATE = .05
  7. Set FICA = .0762
  8. Set netPay = ((grossPay * FED) + (grossPay * STATE) + (grossPay * FICA)) - grossPay
  9. Display "Your Gross pay is " grossPay"."
  10. Display "Would you like for me to take anything else out of your check?"
  11. Input "yes" or "no"
  12. Display "How much would you like to take out of your check?"
  13. Input deductions
  14. Set grossFinal = grossPay - additionalCuts
  15. Display, "Your final pay for the year is " grossFinal"."
  16. End
